My journey through the iconic book **"Fluent Python"** by Luciano Ramalho. This repository is a practical logbook where I solve exercises, break down complex concepts, and write clean, Pythonic code as a beginner Data Science student.

> **Expect mistakes, clean code attempts, and lots of learning. Open to feedback and collaboration!**

---

## 📚 About This Repository

This is not just a collection of solutions; it's a documented learning adventure. The goal is to move beyond theory and truly **apply** the powerful and elegant features of Python to become a more effective Data Scientist.

### What you'll find here:
- **📝 Chapter-wise Exercises:** Solutions and attempts for exercises from "Fluent Python".
- **📒 Personal Notes:** Jargon-free breakdowns of complex topics like decorators, descriptors, and metaclasses.
- **🧪 Experimentation:** Small scripts and test files used to understand how Python works under the hood.
- **🤝 Open for Feedback:** I welcome all kinds of feedback, from bug reports to suggestions for more Pythonic code.

The repository is organized by the book's core concepts to make navigation intuitive.

| Directory | Focus Area | Status |
